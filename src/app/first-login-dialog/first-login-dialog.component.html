First time login in?

How to use this app you ask yourself? well, that's easy:


You have two main buttons, one is Change courses that will let you change your courses.


If you are a student it will apply the courses you want someone to help you with.


If you are a tutor it will apply the courses you can help someone with.


The same with Change days it will let you change the days and hours you can help or get help at.


IMPORTANT! if changing days on mobile, make sure to rotate the phone to see the full scheduler!


Good luck!
